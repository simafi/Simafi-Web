🧮 Ejemplo Visual - Cálculo Factor × Unidad

📋 Cómo Funciona el Cálculo

El sistema calcula el impuesto usando la fórmula: Factor × Unidad

Luego aplica las tarifas ICS sobre el resultado obtenido.

🎯 Ejemplo Práctico

Escenario: Un negocio tiene 500 unidades de un producto con un factor de 1.5

Factor (1.5) × Unidad (500) = 750

El sistema calcula el impuesto sobre el valor 750 usando las tarifas ICS.
