El Tipo de Asiento indica la clase de operación contable y se usa para:
  • Numeración: el tipo tiene un prefijo que ayuda a identificar el origen del asiento.
  • Automatización: algunos tipos están marcados como “automáticos”; el sistema puede actuar distinto según esa configuración.

- Para asientos normales (registro manual): Asiento Manual.
- Para ajustes contables: Asiento de Ajuste.
- Si el sistema genera el asiento: Asiento Automático (si aplica).
