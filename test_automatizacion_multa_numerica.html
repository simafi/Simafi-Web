🔢 Test Automatización Multa - Validación Numérica

Reglas Numéricas:

  • mes === 1 (Enero) → multa = 0.00
  • mes !== 1 && tipo === 1 (≠ Enero + Normal) → multa = impuesto
  • mes !== 1 && tipo !== 1 (≠ Enero + ≠ Normal) → multa = 0.00
  • impuesto <= 0 → multa = 0.00 (sin importar mes/tipo)
