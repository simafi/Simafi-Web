✅ Test: URL de Logout Corregida

Error Corregido: Page not found (404) - tributario-app/login/

Causa: URL incorrecta en la redirección del logout

Solución: Uso de nombre de URL en lugar de hardcodear la ruta

❌ Error 404 Anterior
🐛 Problema de URL:

URL Incorrecta:

def logout_view(request): logout(request) return redirect('/tributario-app/login/') # ❌ URL NO EXISTE

Error: Page not found (404)

Razón: La URL /tributario-app/login/ no está definida en urlpatterns

📋 URLs Disponibles:
✅ login/ [name='login'] ✅ [name='tributario_login'] (ruta raíz) ❌ tributario-app/login/ (NO EXISTE)
✅ Solución Aplicada
🔧 URL Corregida:

Versión Final (Mejor Práctica):

def logout_view(request): logout(request) return redirect('tributario_app:login') # ✅ USA NOMBRE DE URL

Beneficios:

  • ✅ Usa el nombre de la URL
  • ✅ Más mantenible
  • ✅ Funciona correctamente
📋 Configuración de URLs:
# tributario_app/urls.py path('', login_view, name='tributario_login') path('login/', login_view, name='login') # Namespace: tributario_app # Acceso: 'tributario_app:login'
