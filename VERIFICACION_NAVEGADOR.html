🔍 VERIFICACIÓN REAL DEL CÁLCULO UNIDAD × FACTOR

Esta página te permite verificar si el sistema está funcionando correctamente

⚠️ IMPORTANTE:
1. El formulario real está cargado abajo en el iframe
2. Abre la consola del navegador (F12) para ver los logs del sistema
3. Si ya tenías el formulario abierto, recarga con Ctrl+F5

📋 INSTRUCCIONES DE PRUEBA

Paso 1: Verificar Inicialización

  1. Presiona F12 para abrir Developer Tools
  2. Ve a la pestaña Console
  3. Busca el mensaje:
    🔧 Variables ocultas creadas: {...}
  4. Verifica que contenga:
    unidadFactor_impuesto: 0

Paso 2: Probar el Cálculo

  1. En el formulario abajo, busca el campo "Unidad"
  2. Ingresa el valor: 1000
  3. Presiona Tab o haz clic fuera
  4. Busca el campo "Factor"
  5. Ingresa el valor: 5.50
  6. Presiona Tab o haz clic fuera

Paso 3: Verificar en Console

Después de ingresar los valores, debes ver en la consola:

📝 EVENT INPUT DISPARADO PARA: unidad con valor: "1000"
🔢 CALCULANDO IMPUESTOS PARA CAMPO: unidad

🧮 Cálculo Factor × Unidad:
Factor: 5.5
Unidad: 1000
Resultado: 5.5 × 1000 = 5500

📊 Unidad × Factor: 1000 × 5.5 = L. 5500.00
💰 TOTAL IMPUESTO FINAL: L. 5500.00
✅ Campo impuesto actualizado: ... Valor nuevo: L. 5500.00

Paso 4: Verificar en el Formulario

El campo "Impuesto Calculado" o "Impuesto" debe mostrar:

L. 5,500.00

📊 Comandos para la Consola (Copiar y Pegar)

Puedes pegar estos comandos en la consola del navegador para hacer verificaciones:

Verificar que los campos existen:

document.getElementById('id_unidad')
document.getElementById('id_factor')
document.getElementById('id_impuesto')

Cada uno debe retornar un elemento HTML, no null

Verificar valores actuales:

console.log('Unidad:', document.getElementById('id_unidad').value)
console.log('Factor:', document.getElementById('id_factor').value)
console.log('Impuesto:', document.getElementById('id_impuesto').value)

Verificar sistema:

window.declaracionVolumenInteractivo
window.declaracionVolumenInteractivo.variablesOcultas

Debe mostrar el objeto del sistema y sus variables ocultas

Forzar recálculo manual:

window.declaracionVolumenInteractivo.calcularEnTiempoReal('unidad')
